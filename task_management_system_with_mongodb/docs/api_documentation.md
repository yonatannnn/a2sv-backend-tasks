# Task Management API

## Endpoints

### GET /tasks
- Retrieves all tasks.

### GET /tasks/:id
- Retrieves a task by ID.

### POST /tasks
- Creates a new task.
- Body: `{ "title": "Task Title", "description": "Task Description", "completed": false }`

### PUT /tasks/:id
- Updates a task by ID.
- Body: `{ "title": "Updated Title", "description": "Updated Description", "completed": true }`

### DELETE /tasks/:id
- Deletes a task by ID.
